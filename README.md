🧠 Python Developer Quiz Game (CLI)

An interactive command-line based quiz application developed using Python for beginners and fresher developer interview preparation.

The quiz presents multiple-choice questions related to Python fundamentals, operators, loops, functions, modules, and file handling concepts.
Each time the quiz starts, a random set of 30 questions is displayed along with a 30-minute countdown timer, creating an experience similar to a real online assessment.

🚀 Features
🎯 Random question selection every time
⏱ 30-minute timer with auto-submit
🔤 Multiple-choice answer system (A / B / C / D)
📊 Score, percentage, and performance grade
🎨 Colorful and interactive terminal interface
🔁 Replay option for continuous practice
💡 Useful for Python interview preparation

🛠 Technologies Used

Python
Random Module
Time Module
Colorama Library

⚙️ Installation & Setup

Step 1 — Clone Repository
github clone https://github.com/sakshikadam1501-rgb/Python-Developer-quiz-game

Step 2 — Install Required Library
pip install colorama

▶ How to Run

python quiz.py

🎮 Gameplay Flow

1) Quiz starts with welcome screen
2) System randomly selects 30 questions
3) Timer countdown begins (30 minutes)
4) Player selects answers using A / B / C / D
5) Quiz auto-submits if time ends
6) Final score, percentage, and grade displayed
